# MyFramework Documentation

Welcome to **GQFusion-program-framework**, a cross-platform UI + rendering + tooling ecosystem designed for:

- UI development
- Real-time theme systems
- Live configuration editing
- Animation systems
- Multi-backend rendering (Vulkan/OpenGL/Metal)
- Developer tooling (Inspector, Editors, Compiler tools)

---

## 🚀 Core Philosophy

MyFramework is built around:

- Live reloading everything
- Modular subsystems
- Cross-platform abstraction
- Tool-first development (like Unreal/Qt)

---

## 📦 System Overview

- Core → Application + events + object system
- UI → Widgets, layouts, windows
- Render → GPU abstraction layer
- Theme → dynamic visual styling system
- Config → live-editable configuration system
- Animation → timeline + easing engine
- Platform → OS window backends
